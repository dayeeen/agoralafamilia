# Agora La Familia

**Agora La Familia** adalah website reuni alumni SMPN 2 Panumbangan yang berfungsi sebagai ruang temu digital untuk berbagi informasi acara, dokumentasi kenangan, dan mempererat silaturahmi antar alumni.

Website ini dirancang dengan tampilan modern, interaktif, dan responsif agar nyaman diakses di berbagai perangkat.
